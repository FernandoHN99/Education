*************Criando Ambiente Virtual*************

1-) Criar uma pasta/diretório qualquer

2-) Abrir cmd nessa pasta

3-) Instalar virtual virtualenv (Somente necessário uma vez):
		pip install virtualenv

4-) Criar uma Sub-Pasta para o Ambiente Virtual
		py -m venv ./NomeDaSubPasta

5-) Entrar dentro da sub-pasta "Script" criada
		cd /NomeDaSubPasta
		cd /Scrripts

6-) Ativar o activate.bat
		activate.bat
	
7-) Ambiente Virtual Criado Ativado!
		Vai estar com a indicação (venv) no terminal
